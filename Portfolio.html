Single Cycle Processor

I designed a fully functional single-cycle processor using ARMv8. To do this, I developed all the intermediate parts of the processor such as the ALU, sign extender, register file, and memory components.

Cryptarithmetic Solver

A cryptarithmetic puzzle is a mathematical exercise where letters represent a unique digit that is part of a greater number. The goal is to figure out the digit which the letter represents. With this concept in mind, I wrote a python script which uses logical constraints to solve these puzzles. One simple example is the following: if the letter being solved for begins a word, it cannot be 0 because no number starts with 0. This script was created to practice the idea of logical constraints and how they can be applied to artificial intelligence.

Traffic Light Controller

I created a small scale traffic light system using a Raspberry Pi, Python, a button, and a breadboard. The lights would begin with one green intersection and one red. Whenever the button was pressed, the green would flash for 4 seconds and then the lights would switch colors. The new lights would stay for 10 seconds, and then switch back.
