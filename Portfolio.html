MITS

(Multi-Instrumental Tactile Synthesizer)

Multi-Instrumental-Tactile-Synthesizers, or MITS are a dynamically programmable pair of gloves which can play and record multiple instruments and vocals to create music. These work with a DAW (Digital Audio Workstation) like GarageBand, Pro Tools or Reaper. This allows the user to play a multitude of different instruments that are already present in the DAW. The gloves connect to a computer where the audio is generated. They are battery powered also, allowing the user to walk around while creating music.

Receipt Reader

Keeping track of purchases can prove difficult. Receipt Reader allows users to input pictures of their receipts and receive a google spreadsheet containing the receipts store location, purchase date, total cost, and the last four digits of the credit card used.

To do this, the user will enter the phone number. Using Nexmo, the user's identity is verified and their phone number saved to later connect it to a specific google sheet to reference future receipt additions. Next, the user uploads one or multiple receipt photos which will be OCRed using Google Vision OCR, processed in the backend using javascript, and then placed in a google sheet. The user then enters their email and the spreadsheet’s shareable link which is sent to them via email. If the user is new, the spreadsheet will now be linked to the user’s phone number so that future receipts can be added to the same sheet.

This app not only allows users an easy way to compile and store their finance information, but it gives them the opportunity to gain valuable insights from their purchases - where they are spending their money, when, how much, and with what card. As data becomes more important, this receipt reader is not only priceless in itself, but in the long run can save money!
